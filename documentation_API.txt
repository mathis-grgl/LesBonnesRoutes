1- Création d’un compte utilisateur (inscription)
    PUT api/v1/user
        HEAD: /
        BODY: mail, password, nom, prenom, tel

        reponses: 
            201 (created) compte crée
            400 (bad request) information éronnées ou manquante
            409 (conflict) le compte existe déjà

        reponse_content: token d'authentification

2- Modification d’un compte utilisateur
    PATCH api/v1/user/{user_id}
        HEAD: auth_token
        BODY: [champ(s) a modifier], password

        reponses:
            200 (ok)
            400 (bad request) information éronnées ou manquante
            401 (unauthorized) token invalide ou expiré

        reponse_content: [champs modifiés sauf password], nouveau token de connection (si nouveau mot de passe)

3- Suppression d’un compte utilisateur
    DELETE api/v1/user/{user_id}
        HEAD: auth_token
        BODY: /

        reponses:
            204 (no content)
            401 (unauthorized) token invalide ou expiré 
            403 (forbidden) le token n'est pas admin et le compte a supprimer n'est pas celui de l'utilisateur

        reponse_content: /

4- Se connecter
    POST api/v1/user/connect
        HEAD: /
        BODY: mail + password
        
        reponses:
            200 (ok)
            401 (unauthorized) mail ou mot de passe invalide

        reponse_content: token d'authentification
